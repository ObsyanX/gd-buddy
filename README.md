## 🎯 What is GD Buddy?

**[GD Buddy](https://gd-buddy.vercel.app)** is a free, AI-powered **group discussion practice platform** designed for students, job-seekers, and campus placement aspirants preparing for **GD rounds** in interviews.

It creates a **virtual panel of AI participants** — each with unique personalities, debate styles, and viewpoints — allowing you to practice realistic group discussions anytime, anywhere.

**→ [Start Practicing Free](https://gd-buddy.vercel.app)**

## 🌟 Key Features

### 🤖 Multi-Persona AI Participants
- Different personalities: *Aggressive Debater, Logical Thinker, Peacekeeper, Data-Driven Speaker, Devil's Advocate*
- Each contributes naturally with interruptions, counterpoints, and follow-ups
- 2–6 AI participants per session

### 🎯 50+ Curated GD Topics
- Categories: *Current Affairs, Technology, Business, Social Issues, Abstract*
- [Browse all GD topics for placements](https://gd-buddy.vercel.app/gd-topics-for-placements)
- AI-generated custom topics on demand

### 🎙️ Voice-Based Discussion
- Speak naturally using your microphone
- Real-time speech-to-text transcription
- AI participants respond with realistic voices

### 📊 Detailed Performance Reports
- Content quality, fluency, and structure scores
- Filler word count and speaking pace (WPM)
- Eye contact, posture, and body language analysis (via webcam)
- Actionable improvement suggestions

### 🎭 Persona Customization
- Choose number of participants (2–6)
- Customize tone, verbosity, agreeability, and expertise
- Create and save custom personas

### 👥 Multiplayer Mode
- Create a room and share the code with friends
- Practice together alongside AI participants
- Real-time presence and turn-taking

### 🎯 Skill Drills
- Targeted practice: Opening Statements, Rebuttals, STAR Responses, Time-Boxed Arguments
- AI-generated feedback on each drill

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ or Bun
- Supabase project (for backend)

### Clone and Run

```bash
git clone https://github.com/ObsyanX/gd-buddy.git
cd gd-buddy
npm install
npm run dev
```

Visit `http://localhost:5173` to start practicing.
